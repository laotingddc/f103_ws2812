# WS2812 LED串口控制协议

## 通信参数
- 波特率: 115200
- 数据位: 8
- 停止位: 1
- 校验位: 无

## 帧格式

```
| 帧头 | 命令 | 参数... | 校验和 |
| 0xAA | CMD  | DATA    | SUM    |
```

- 帧头: 固定为 `0xAA`
- 命令: 1字节命令码
- 参数: 根据命令不同，长度不同
- 校验和: 从命令字节开始到校验和前一字节的累加和(取低8位)

## 命令列表

### 0x01 - 设置单个LED颜色
设置指定索引的LED颜色。

| 字节 | 内容 | 说明 |
|------|------|------|
| 0 | 0xAA | 帧头 |

| 2 | IDX_L | LED索引低字节 |
| 3 | IDX_H | LED索引高字节 |
| 4 | R | 红色 (0-255) |
| 5 | G | 绿色 (0-255) |
| 6 | B | 蓝色 (0-255) |
| 7 | SUM | 校验和 |

### 0x0A - 设置单路GPIO
设置指定GPIO通道输出电平（16路）。

通道映射（CH0~CH15）：
- CH0: PA2
- CH1: PA3
- CH2: PA4
- CH3: PA5
- CH4: PA6
- CH5: PA7
- CH6: PA8
- CH7: PB0
- CH8: PB1
- CH9: PB2
- CH10: PB10
- CH11: PB11
- CH12: PB12
- CH13: PB13
- CH14: PB14
- CH15: PB15

电平逻辑：`1=高电平(开)`，`0=低电平(关)`。

| 字节 | 内容 | 说明 |
|------|------|------|
| 0 | 0xAA | 帧头 |
| 1 | 0x0A | 命令 |
| 2 | CH_L | 通道索引低字节 |
| 3 | CH_H | 通道索引高字节 |
| 4 | VAL | 输出值(0/1) |
| 5 | SUM | 校验和 |

### 0x0B - 设置16路GPIO位图
一次性设置16路GPIO状态。

| 字节 | 内容 | 说明 |
|------|------|------|
| 0 | 0xAA | 帧头 |
| 1 | 0x0B | 命令 |
| 2 | MASK_L | 位图低字节 |
| 3 | MASK_H | 位图高字节 |
| 4 | SUM | 校验和 |

说明：
- bit0 对应 CH0，bit15 对应 CH15
- bit=1 表示该通道输出高电平，bit=0 表示低电平

### 0x0C - 关闭所有GPIO
将16路GPIO全部输出低电平。

| 字节 | 内容 | 说明 |
|------|------|------|
| 0 | 0xAA | 帧头 |
| 1 | 0x0C | 命令 |
| 2 | SUM | 校验和 |

## 校验和计算示例

设置第0个LED为红色 (R=255, G=0, B=0):
```
数据: 0x01 0x00 0x00 0xFF 0x00 0x00
校验和 = 0x01 + 0x00 + 0x00 + 0xFF + 0x00 + 0x00 = 0x00 (取低8位)
完整帧: AA 01 00 00 FF 00 00 00
```

设置所有LED为绿色 (R=0, G=255, B=0):
```
数据: 0x02 0x00 0xFF 0x00
校验和 = 0x02 + 0x00 + 0xFF + 0x00 = 0x01
完整帧: AA 02 00 FF 00 01
```

设置GPIO通道0为高电平:
```
数据: 0x0A 0x00 0x00 0x01
校验和 = 0x0A + 0x00 + 0x00 + 0x01 = 0x0B
完整帧: AA 0A 00 00 01 0B
```

设置GPIO位图为0x00FF (CH0~CH7高电平):
```
数据: 0x0B 0xFF 0x00
校验和 = 0x0B + 0xFF + 0x00 = 0x0A (取低8位)
完整帧: AA 0B FF 00 0A
```

## 硬件连接
- LED数据线: PB6 (TIM4_CH1)
- 串口TX: PA9
- 串口RX: PA10
- LED数量: 96个
