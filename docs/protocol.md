# WS2812 LED串口控制协议

## 通信参数
- 波特率: 115200
- 数据位: 8
- 停止位: 1
- 校验位: 无

## 帧格式

```
| 帧头 | 命令 | 参数... | 校验和 |
| 0xAA | CMD  | DATA    | SUM    |
```

- 帧头: 固定为 `0xAA`
- 命令: 1字节命令码
- 参数: 根据命令不同，长度不同
- 校验和: 从命令字节开始到校验和前一字节的累加和(取低8位)

## 命令列表

### 0x01 - 设置单个LED颜色
设置指定索引的LED颜色。

| 字节 | 内容 | 说明 |
|------|------|------|
| 0 | 0xAA | 帧头 |
| 1 | 0x01 | 命令 |
| 2 | IDX_L | LED索引低字节 |
| 3 | IDX_H | LED索引高字节 |
| 4 | R | 红色 (0-255) |
| 5 | G | 绿色 (0-255) |
| 6 | B | 蓝色 (0-255) |
| 7 | SUM | 校验和 |

## 校验和计算示例

设置第0个LED为红色 (R=255, G=0, B=0):
```
数据: 0x01 0x00 0x00 0xFF 0x00 0x00
校验和 = 0x01 + 0x00 + 0x00 + 0xFF + 0x00 + 0x00 = 0x00 (取低8位)
完整帧: AA 01 00 00 FF 00 00 00
```

设置所有LED为绿色 (R=0, G=255, B=0):
```
数据: 0x02 0x00 0xFF 0x00
校验和 = 0x02 + 0x00 + 0xFF + 0x00 = 0x01
完整帧: AA 02 00 FF 00 01
```

## 硬件连接
- LED数据线: PB6 (TIM4_CH1)
- 串口TX: PA9
- 串口RX: PA10
- LED数量: 96个
